\subsubsection[CPU Options]{CPU OPTIONS}

These are the currently supported "normal" CPU options:

faultbase

num

data

stopnum

mode

speed

port

assignment

interlace

enable

init\_enable

store\_size

These options are supported as "hacks" and should not normally be used:

dis\_enable

halt\_on\_unimplemented

disable\_wam

report\_faults

tro\_enable

drl\_fatal

useMap

address

disable\_cache
